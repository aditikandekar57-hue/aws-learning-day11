# AWS Application Load Balancer Project

## Description
Implemented an Application Load Balancer to distribute traffic across multiple EC2 instances.

## Architecture
User → ALB → EC2 Instances

## Steps
- Launched 2 EC2 instances
- Installed Nginx web server
- Created target group
- Configured Application Load Balancer
- Verified traffic distribution
